# KoreaBridge

KoreaBridge is a trusted start website for foreigners in Korea.

It combines:

- a GaijinPot-style life hub for housing, work, guides, and neighborhoods
- a Go! Go! Nihon-style first-90-days arrival path
- an Integreat-style official guide layer
- a Uniplaces-style housing confidence model
- a transparent room and job risk checker

## Live Website

This repository is intended to be published with GitHub Pages from the `main` branch root.

## Local Files

- `index.html`: website structure
- `styles.css`: responsive visual system
- `app.js`: interactive website behavior
- `risk-engine.js`: housing and job risk scoring engine

## Product Promise

Check your room. Check your job. Start Korea safely.

The local workspace includes Node tests for:

- risk scoring boundaries
- critical housing and job risk scenarios
